Q1 选择题 复习题·一-1
在空间直角坐标系下，函数 z = 2026 − x² + y² 的图像为（ ）。
A. 球面 B. 圆柱面 C. 抛物面 D. 圆锥面
二次曲面 双曲抛物面（马鞍面） 曲面方程识别
▼ 点击查看答案与解析
✅ 答案：C（抛物面）——具体为双曲抛物面（马鞍面）
📝 解析：方程 z = 2026 − x² + y² 可写为标准形式 z = −x² + y² + 2026。该方程中 x² 和 y² 的系数符号相反（一负一正），这是双曲抛物面（马鞍面）的典型特征。当截 z = k 时得到双曲线，截 x = 0 得 z = y² + 2026（开口向上的抛物线），截 y = 0 得 z = −x² + 2026（开口向下的抛物线）。
💡 方法总结：识别二次曲面方程的关键：①椭球面：x²、y²、z² 系数同号；②抛物面（椭圆抛物面）：缺少一个平方项，另两个同号；③双曲抛物面：缺少一个平方项，另两个异号（或 z 为另两个的差）；④锥面：三个平方项和为0（如 x²+y²−z²=0）；⑤柱面：缺少一个变量。
Q2 选择题 复习题·一-2
在空间直角坐标系下，方程 z = 2026 − x² − y² 的图像为（ ）。
A. 球面 B. 圆柱面 C. 圆锥面 D. 抛物面
二次曲面 椭圆抛物面 旋转抛物面
▼ 点击查看答案与解析
✅ 答案：D（抛物面）——具体为椭圆抛物面（旋转抛物面）
📝 解析：方程 z = 2026 − x² − y² 即 z − 2026 = −(x² + y²)。x² 和 y² 的系数同号（均为负），这是椭圆抛物面的特征。由于 x² 和 y² 系数相等，实际上是旋转抛物面（由抛物线绕 z 轴旋转而成），开口向下，顶点在 (0,0,2026)。
💡 方法总结：椭圆抛物面的标准形式：z = x²/a² + y²/b²（开口向上）或 z = −x²/a² − y²/b²（开口向下）。当 a = b 时为旋转抛物面。与马鞍面（Q1）的区别在于：马鞍面中 x² 和 y² 系数异号。
Q3 选择题 复习题·一-3
在空间直角坐标系下，方程 x² − y² + z² = 2026 的图像为（ ）。
A. 单叶双曲面 B. 双叶双曲面 C. 球面 D. 圆锥面
二次曲面 单叶双曲面 双叶双曲面
▼ 点击查看答案与解析
✅ 答案：A（单叶双曲面）
📝 解析：方程 x² − y² + z² = 2026 写成标准形式：x²/2026 − y²/2026 + z²/2026 = 1。三个平方项中有一个为负（y² 项），且等式右边为正——这是单叶双曲面的标志。若有两个平方项为负或右边为-1，则为双叶双曲面。
💡 方法总结：单叶双曲面 vs 双叶双曲面的关键判断：①方程化为 =1 形式后，负平方项个数为 1 → 单叶双曲面；②负平方项个数为 2（或 =−1 形式后正项为 1）→ 双叶双曲面。单叶双曲面是连通的，双叶双曲面分为两片。
Q4 选择题 复习题·一-4 模拟卷二·一-4
直线 L: (x−2)/3 = (y+2)/1 = (z−3)/(−4) 和平面 Π: x + y + z = 3 的位置关系为（ ）。
A. L 在 Π 内 B. L 与 Π 相交 C. L 与 Π 平行 D. 以上均不正确
直线与平面的位置关系 方向向量 法向量 点积判断
▼ 点击查看答案与解析
✅ 答案：B（L 与 Π 相交）
📝 解析：直线的方向向量 s = (3, 1, −4)，平面的法向量 n = (1, 1, 1)。计算 s · n = 3×1 + 1×1 + (−4)×1 = 3 + 1 − 4 = 0。由于方向向量与法向量点积为零，说明 s ⊥ n，即直线与平面平行。但还需验证是否在平面内：取直线上一点 (2, −2, 3)，代入平面方程得 2 + (−2) + 3 = 3 = 3 ✓，说明该点在平面上，因此直线在平面内。但选项只有"相交、平行、在内"，由于点在平面内 → 直线在平面内，选 A？等一下——选项 A 是"L 在 Π 内"。方向向量与法向量垂直 + 点在平面上 = 直线在平面内。

再仔细验证：L 上一点为 (2, −2, 3)，代入平面得 2 + (−2) + 3 = 3，成立。s · n = 0。故 L 在 Π 内，选 A。
💡 方法总结：直线与平面位置关系三步法：①求直线方向向量 s 和平面法向量 n；②计算 s · n——若不为0则相交，若为0则平行或直线在平面内；③取直线上一点代入平面方程——若成立则在平面内，否则平行。
Q5 选择题 复习题·一-5
设直线 L 的方程为 x − y + z = 1, 2x + y + z = 4（一般方程），则 L 的参数方程为（ ）。
A. x = 1−2t, y = 1+t, z = 1+3t B. x = 1−2t, y = −1+t, z = 1+3t C. x = 1−2t, y = 1−t, z = 1+3t D. x = 1−2t, y = −1−t, z = 1+3t
直线的一般方程 参数方程转换 方向向量求解
▼ 点击查看答案与解析
✅ 答案：A
📝 解析：由一般方程转化：两平面的法向量分别为 n₁ = (1, −1, 1) 和 n₂ = (2, 1, 1)。直线方向向量 s = n₁ × n₂ = (−2, 1, 3)（计算：|i j k; 1 −1 1; 2 1 1| = i(−2) − j(−1) + k(3) = (−2, 1, 3)）。取直线上一点：令 z = 1，解 x − y = 0, 2x + y = 3 → x = 1, y = 1。故参数方程为 x = 1 − 2t, y = 1 + t, z = 1 + 3t，选 A。
💡 方法总结：一般方程 → 参数方程：①方向向量 = 两平面法向量的叉积；②取直线上一点（令某一变量为常数解方程组）；③写出参数方程。

下列结论中，错误的是（ ）。
A. z + 2x² + y² = 0 表示椭圆抛物面 B. x² + 2y² = 1 + 3z² 表示双叶双曲面 C. x² + y² − (z−1)² = 0 表示圆锥面 D. y² = 5x 表示抛物柱面

Q7 选择题 复习题·一-7
曲线 x² + y² + z² = 1, x + z = 1 在 xOy 平面上的投影曲线的方程是（ ）。
空间曲线投影 消去变量法 投影曲线方程
▼ 点击查看答案与解析
✅ 答案：A —— 2x² − 2x + 2y² = 0（或 (x−1/2)² + y² = (1/2)²）, z = 0
📝 解析：由 x + z = 1 得 z = 1 − x，代入球面方程：x² + y² + (1−x)² = 1 → x² + y² + 1 − 2x + x² = 1 → 2x² − 2x + y² = 0 → 2(x² − x + 1/4) + y² = 1/2 → 2(x − 1/2)² + y² = 1/2。投影曲线需加 z = 0。整理得 4(x − 1/2)² + 2y² = 1, z = 0。
💡 方法总结：求空间曲线在坐标面上的投影：①消去非投影轴对应的变量（如投影到 xOy 平面则消去 z）；②加上投影面的方程（z = 0）；③化为标准形式。
Q8 选择题 复习题·一-8
曲线 x² + 4y² − z² = 16, 4x² + y² + z² = 4 在 xOy 平面上的投影曲线方程是（ ）。
空间曲线投影 方程组消元
▼ 点击查看答案与解析
✅ 答案：C —— x² + y² = 4, z = 0
📝 解析：将两式相加消去 z²：(x² + 4y² − z²) + (4x² + y² + z²) = 16 + 4 → 5x² + 5y² = 20 → x² + y² = 4。加 z = 0 得投影方程。
💡 方法总结：两个曲面交线的投影：通过加减消元消去非投影轴变量，得到仅含投影平面变量的方程，再补充投影面方程即可。
Q9 选择题 复习题·一-9
双曲线 x²/3 − z²/4 = 1, y = 0 绕 z 轴旋转一周所得的曲面方程为（ ）。
旋转曲面 绕z轴旋转 旋转单叶双曲面
▼ 点击查看答案与解析
✅ 答案：A —— x²/3 + y²/3 − z²/4 = 1（旋转单叶双曲面）
📝 解析：曲线在 xOz 平面（y = 0），绕 z 轴旋转时，z 坐标不变，而 x² 被替换为 x² + y²（因为原曲线上的点到旋转轴的距离为 |x|，旋转后变为 √(x²+y²)）。将原方程 x²/3 − z²/4 = 1 中的 x² 替换为 x² + y²，得 (x²+y²)/3 − z²/4 = 1。这是一个旋转单叶双曲面。
💡 方法总结：旋转曲面方程：曲线在坐标平面内绕坐标轴旋转——①绕 z 轴旋转：将方程中另一变量（x 或 y）的平方替换为 x² + y²；②绕 x 轴旋转：将其他变量替换为 y² + z²；③绕 y 轴旋转：将其他变量替换为 x² + z²。
Q10 填空题 复习题·二-1

▼ 点击查看答案与解析

Q12 填空题 复习题·二-3

Q13 填空题 复习题·二-4

▼ 点击查看答案与解析
✅ 答案：x − 3y − 2z = 0

Q16 填空题 模拟卷二·三-17
